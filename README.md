# 🎮 Gamified 3D Portfolio | Nikhil Jangid

A unique, interactive web developer portfolio built like a 3D escape room. Instead of just scrolling through a standard webpage, visitors must solve tech-based logic puzzles to unlock different "Archives" and reveal information about my skills, projects, and contact details.

## ✨ Core Features

* **🚪 3D Interactive Room:** A symmetrical 3D environment that subtly shifts perspective based on mouse movement.
* **🧩 Logic Puzzles:** Doors are locked by default. Visitors must correctly answer riddles related to HTML, JavaScript, and tech platforms to progress.
* **🎨 Canvas Animations:** Features custom background ink particles and a celebratory confetti engine built with the HTML5 `<canvas>` API.
* **🖱️ Custom UI:** Includes a gamified custom cursor, custom scrollbars, and interactive ripple click effects.

## 🛠️ Tech Stack

This project is built purely with core web fundamentals (No external frameworks or libraries):
* **HTML5:** Semantic structure and modal containers.
* **CSS3:** 3D transforms (`preserve-3d`, `rotateY`), custom animations, and responsive design.
* **Vanilla JavaScript:** DOM manipulation, game state management, event listeners, and particle physics.
